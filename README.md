# AI-Interior-Design-Assistant
It is an (DesignSense AI) interior design platform built with Base44, helping users to visualize and create stunning interior designs.

## Features
-AI powered design suggestions
-Customizable room layouts
-Furniture and decor library
-Lighting optimization
-Colour suggestion

## Live Demo
Checkout the live app: https://a5-interiors-design.base44.app

## Built with:
Base44 -- A no code platform.

AI-Interior-Design-Assistant showcases the power of no-code tools for rapid prototyping and innovation.
